# 📘 Assignment: Interactive Web Pages with JavaScript

## 🎯 Objective

Build dynamic, interactive web pages using JavaScript by mastering DOM manipulation, event handling, and real-time user interactions to create engaging web experiences.

## 📝 Tasks

### 🛠️ Task 1: Build an Interactive Todo List

#### Description
Create a todo list application where users can add, display, and remove tasks dynamically. The list should update the DOM in real-time without page refreshes, allowing users to manage their tasks interactively.

#### Requirements
Completed program should:

- Allow users to enter a task in an input field and add it to the list
- Display all tasks in an unordered list on the page
- Include a delete button for each task that removes it from the list
- Update the task count showing total number of tasks
- Prevent adding empty tasks to the list


### 🛠️ Task 2: Create a Form Validator

#### Description
Build a form validator that checks user input in real-time and provides immediate feedback. Users should see validation messages appear as they type, improving the user experience.

#### Requirements
Completed program should:

- Validate that email field contains a valid email format
- Validate that password is at least 8 characters long
- Check that password and confirm password fields match
- Display error messages in red when validation fails
- Display success messages in green when validation passes
- Update validation status as the user types (real-time feedback)


### 🛠️ Task 3: Build an Interactive Counter

#### Description
Create an interactive counter application with buttons to increment, decrement, and reset the count. Practice working with event listeners and updating the DOM based on user actions.

#### Requirements
Completed program should:

- Display the current count on the page
- Include "Increment" button that increases count by 1
- Include "Decrement" button that decreases count by 1
- Include "Reset" button that sets count back to 0
- Apply styling that changes color based on count (green for positive, red for negative, neutral for zero)
- Prevent the count from going below -10 or above 10
